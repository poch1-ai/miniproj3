Manipulating strings

str functions to know for manipulating strings:

  • str_length(): Count how many characters are there in the string
  • str_sub(): Pull out character base on the pocision
  • str_c(): to combine all the character string
  • str_to_lower(): everything to lowercase
  • str_to_upper(): Everything upper case
  • str_to_title():make the first letter capital
  • str_replace() not in 02_strings examples
